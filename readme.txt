== Description ==

SoyebSalar Dynamic Block is a custom Gutenberg block for WordPress that allows you to create dynamic content including an image, title, subtitle, and button. Customize the block with settings for image size and alignment, title and subtitle font styles, button size, and alignment.

== Features ==

* Add an image with customizable size and alignment.
* Set the title and subtitle with adjustable font size, font family, and font style.
* Add a button with customizable text, size, position, and URL.
* Fully compatible with the Gutenberg editor.
* Easy to use and customize.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ssgb-dynamic-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Create or edit a post or page, and use the SoyebSalar Dynamic Block within the Gutenberg editor.

== Frequently Asked Questions ==

= How do I change the number of columns? =

Currently, the block is set to display content in a single layout. If you need additional customization, such as column layouts, you might consider adding custom CSS or modifying the block's code.

= Can I customize the block's styles? =

Yes, you can customize the block's appearance using the provided settings in the Gutenberg editor or by adding custom CSS in your theme or child theme.
